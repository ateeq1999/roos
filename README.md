# roos

**Rust Orchestration Operating System**

ROOS is an open-source, production-grade AI agent orchestration framework written in Rust.

> Documentation: https://docs.roos.dev (planned)

## Quick Start

```bash
cargo install roos-cli
roos new my-agent
cd my-agent
roos run my-agent --input "Hello"
```
